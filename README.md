# bank-customer-segmentation-and-classification

## Project Overview

This project performs customer segmentation using K-Means Clustering and predicts customer segments using Classification Models.

## Objectives
- Identify customer segments
- Analyze customer characteristics
- Build a classification model to predict customer segments

## Dataset Features
- TransactionAmount
- CustomerAge
- TransactionDuration
- AccountBalance
- TransactionType
- Location
- Channel
- CustomerOccupation

## Machine Learning Workflow
1. Data Cleaning
2. Feature Encoding
3. Scaling
4. K-Means Clustering
5. PCA Visualization
6. Cluster Interpretation
7. Decision Tree Classification
8. Model Evaluation

## Algorithms

### Clustering
- K-Means
- PCA

### Classification
- Decision Tree
- Random Forest (Exploration)

## Results

### Cluster 0
Senior customers with higher balances and lower spending tendencies.

### Cluster 1
Younger customers with higher transaction activity and lower balances.
